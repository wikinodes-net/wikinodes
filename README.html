Follow this steps to run Hello Word

  1. Install nix-shell (https://developer.holochain.org/docs/install/)
  2. Clone this repository
  3. Go to root folder where you see default.nix and run nix-shell (It takes a lot of time just for the first time.)
  4. Go to tests folder and run npm i
  5. In the tests folder run npm test
